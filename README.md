# PetNote-oss
A fully SwiftUI-based iOS app to record everything.

## Features
[x] DIY TabView
[x] PageView Card
[x] NukeUI lazy load image
[x] Alamofire request network interface, upload images
[x] 2 ScrollView  interactive
[x] Background publish content
[x] Login
